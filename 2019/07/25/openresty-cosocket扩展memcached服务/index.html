1. Openresty Cosocket 基础

cosocket: cosocket 是 OpenResty 中的专有名词，是把协程和网络套接字的英文拼在一起形成的，即 cosocket = coroutine + socket。
cosocket 使我们可以实现整真意义上的同步非阻塞编程，是各种 lua-resty-* 非阻塞库的基础.

注: 我们可以通过 cosocket 实现四层网络流量的处理。极大的极高的灵活性
TCP 相关的 cosocket API 可以分为下面这几类。

操作名称 函数名
创建对象 ngx.socket.tcp
设置超时 tcpsock:settimeout 和 tcpsock:settimeouts
建立连接 tcpsock:connect
发送数据 tcpsock:receive、tcpsock:receiveany 和 tcpsock:receiveuntil
连接池 tcpsock:setkeepalive
关闭连接 tcpsock:close

3. 其它

碰到的问题:

  1. 在调试 tcp 接受请求时, 无响应且cpu 被打满。
    排查发现，是因为处理网络收包过程中出现的错误, 一直在死循环中, 调用 receive 函数。
